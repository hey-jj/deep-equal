# Changelog

## [0.2.0] - 2026-07-07

### Changed

- Long binary, octal, and hex strings now round like `Number(string)` in loose numeric comparisons (#17).

### Performance

- Large array comparisons use less time and memory (#18).
- Large Set comparisons allocate less memory (#19).
- Large Map comparisons allocate less memory (#20).
